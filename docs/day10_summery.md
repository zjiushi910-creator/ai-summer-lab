## 完成项目

GitHub User Analyzer

功能：

- 获取 GitHub 用户信息
- 输出用户名
- 输出仓库数量
- 输出 Followers
- 输出 Following
- 输出创建时间
- 输出简介
- 处理 None 数据
- 请求异常处理
- logging 日志记录

## 企业开发思想

- 一个方法只负责一个功能
- 使用 OOP 保存对象状态
- 使用 logging 记录程序运行日志
- 使用 try/except 提高程序健壮性
- 使用 main() 作为程序入口
- 配置和数据分离
